## 简介

**量盾（LiangDun）** 是一款轻量级 Windows 系统安全工具，集系统监控、进程管理、病毒扫描、实时防护、网络监控、文件清理等功能于一体。基于 Python + CustomTkinter 构建，界面简洁现代，资源占用低，支持后台托盘运行。

> 官网：[liangdun.top](https://liangdun.top)

## 功能特性

| 模块 | 说明 |
|------|------|
| **系统监控** | 实时展示 CPU、内存、磁盘使用率，含进度条与详细指标 |
| **进程管理** | 查看所有运行进程，支持按 PID 结束进程 |
| **病毒扫描** | 本地文件扫描 + 在线哈希查毒（CIRCL Hashlookup），支持单文件/文件夹/多文件 |
| **实时防护** | 后台持续监控新进程，自动在线查毒，发现威胁弹窗告警并支持一键终止 |
| **网络监控** | 实时展示 TCP/UDP 连接列表，支持搜索过滤与结束连接对应进程 |
| **快速查杀** | 一键扫描临时目录、下载、桌面、AppData 等高危区域 |
| **全盘查杀** | 全盘深度扫描，带进度条、实时路径显示、结果导出 |
| **隔离沙盒** | 将可疑文件移入隔离区，支持清单记录（v4_plus） |
| **文件清理** | 清理系统临时文件、浏览器缓存、回收站 |
| **启动项管理** | 查看用户/系统启动项，支持设置或取消量盾开机自启 |
| **设置记忆** | 白名单、清理选项、防护开关等配置持久化保存 |
| **系统托盘** | 关闭窗口后最小化到托盘，后台静默运行 |

## 快速开始

### 环境要求

- Windows 10 / 11
- Python 3.8+

### 安装依赖

```bash
pip install customtkinter psutil pystray Pillow requests pywin32
```

### 运行

```bash
# 基础版（5 个功能模块）
python v1.py

# 完整版（9 个功能模块）
python v4.py

# 增强版（含隔离沙盒、多文件选择）
python v4_plus.py
```

### 打包为 EXE

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "量盾" --icon=icon.ico v4_plus.py
```

## 在线查毒

量盾使用 [CIRCL Hashlookup](https://circl.lu/services/hashlookup/) 公共服务进行文件哈希在线查询，无需 API Key。查询流程：

1. 计算目标文件的 SHA256 哈希值
2. 向 `https://hashlookup.circl.lu/hash/{sha256}` 发起查询
3. 根据返回结果判断文件是否为已知恶意软件

## 安全说明

- 量盾是一款**辅助安全工具**，不能替代专业杀毒软件
- 实时防护依赖在线哈希查询，网络不可用时仅做本地检测
- 结束系统关键进程可能导致系统不稳定，请谨慎操作
- 隔离沙盒仅做文件移动，不提供加密或沙箱执行环境
