📁 项目结构
────────────────────────────────────────────────────────────────
E:\MySafeTool\
├── src/                 # 源代码目录
│   └── main.py          # 主程序（核心代码）
├── dist/                # 打包后的 EXE 文件存放位置
├── requirements.txt     # Python 依赖库列表
└── README.txt           # 本说明文件

✨ 功能特性
────────────────────────────────────────────────────────────────
✓ 系统实时监控（CPU、内存、磁盘、网络）
✓ 进程管理（查看、结束进程）
✓ 病毒扫描（威胁检测、可疑文件分析）
✓ 文件清理（清理缓存、临时文件、回收站）
✓ 启动项管理
✓ 现代 GUI 界面（customtkinter）
✓ 无需登录，打开即用
✓ 单文件 EXE 打包

🚀 安装步骤
────────────────────────────────────────────────────────────────
1. 进入项目目录：
   cd E:\MySafeTool

2. 安装所有依赖（一键复制安装）：
   pip install -r requirements.txt

   或直接复制下方命令：
   pip install customtkinter psutil pyinstaller

3. 确保已安装 Python 3.8+ 版本

🎮 运行程序
────────────────────────────────────────────────────────────────
方法1：运行源码
  cd src
  python main.py

方法2：直接运行 EXE
  双击 dist\量盾.exe

📦 打包步骤
────────────────────────────────────────────────────────────────
1. 进入项目目录：
   cd E:\MySafeTool

2. 打包成 EXE（一键复制命令）：
   pyinstaller --onefile --noconsole --name 量盾 src/main.py

3. 打包后的 EXE 在 dist\ 文件夹中

⚠️  图标设置（可选）
────────────────────────────────────────────────────────────────
打包时自定义图标（如果有图标文件 icon.ico）：
  pyinstaller --onefile --noconsole --name 量盾 --icon=icon.ico src/main.py

📖 功能说明
────────────────────────────────────────────────────────────────
1. 系统监控
   - 实时 CPU 使用率监控
   - 多核 CPU 详细显示
   - 内存使用情况（总/已用/可用）
   - 磁盘分区使用率
   - 自动2秒刷新

2. 进程管理
   - 实时显示所有运行进程
   - 搜索过滤进程
   - 查看进程资源占用（CPU/内存）
   - 结束选中进程（带确认对话框）

3. 病毒扫描
   - 扫描文件夹中的文件
   - 检查文件大小异常
   - 检查可疑文件扩展名
   - 检查可疑文件名模式
   - 生成文件指纹（前16字节）
   - 提供威胁评分和建议

4. 文件清理
   - 清理系统临时文件
   - 清理浏览器缓存
   - 清空回收站
   - 显示清理统计

5. 启动项
   - 查看当前用户启动项
   - 列出所有自启动程序

⚠️  病毒扫描说明
────────────────────────────────────────────────────────────────
量盾的病毒扫描功能进行的是**威胁检测**，并非完整病毒扫描。

它会检查：
✓ 文件大小是否异常（超大文件）
✓ 文件扩展名是否可疑
✓ 文件名是否包含可疑关键词
✓ 生成文件指纹（前16字节）

⚠️  重要提醒：
1. 这不是专业杀毒软件，仅供参考
2. 如发现严重威胁，请使用专业杀毒软件（如Windows Defender、360等）
3. 对于 .exe, .scr 等可执行文件需特别谨慎
