📁 项目结构
────────────────────────────────────────────────────────────────
E:\MySafeTool\
├── src/                 # 源代码目录
│   └── main.py          # 主程序（核心代码）
├── dist/                # 打包后的 EXE 文件存放位置
├── requirements.txt     # Python 依赖库列表
└── README.txt           # 本说明文件

✨ 功能特性
────────────────────────────────────────────────────────────────
✓ 系统实时监控（CPU、内存、磁盘、网络）
✓ 进程管理（查看、结束进程）
✓ 病毒扫描（威胁检测、可疑文件分析）
✓ 文件清理（清理缓存、临时文件、回收站）
✓ 启动项管理
✓ 现代 GUI 界面（customtkinter）
✓ 无需登录，打开即用
✓ 单文件 EXE 打包

🚀 安装步骤
────────────────────────────────────────────────────────────────
1. 进入项目目录：
   cd E:\MySafeTool

2. 安装所有依赖（一键复制安装）：
   pip install -r requirements.txt

   或直接复制下方命令：
   pip install customtkinter psutil pyinstaller

3. 确保已安装 Python 3.8+ 版本

🎮 运行程序
────────────────────────────────────────────────────────────────
方法1：运行源码
  cd src
  python main.py

方法2：直接运行 EXE
  双击 dist\量盾.exe

📦 打包步骤
────────────────────────────────────────────────────────────────
1. 进入项目目录：
   cd E:\MySafeTool

2. 打包成 EXE（一键复制命令）：
   pyinstaller --onefile --noconsole --name 量盾 src/main.py

3. 打包后的 EXE 在 dist\ 文件夹中

⚠️  图标设置（可选）
────────────────────────────────────────────────────────────────
打包时自定义图标（如果有图标文件 icon.ico）：
  pyinstaller --onefile --noconsole --name 量盾 --icon=icon.ico src/main.py
